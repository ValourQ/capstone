# capstone
capstone project web dev

Project Overview

This project is a simple responsive web interface for a food delivery platform.
The site includes multiple sections such as a navigation bar, hero banner, search bar, category grid, featured restaurants, promotional offers, customer reviews, an app download section, and a footer.

The entire project is built using HTML5 and CSS3 only, following semantic structure, responsive design rules, and clean layout practices.

Features

Semantic HTML structure

Flexbox used for header, hero section, and footer

CSS Grid used for categories and restaurant cards

Responsive layout with media queries

Hover effects for buttons and cards

Clean, modern UI design

Accessibility considerations (alt attributes and skip link)

Organized folder structure with an images/ directory
